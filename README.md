# 桃鉄風ゲーム（ロジック学習用）

## 目的
サイコロを振り、十字キーで出た目の数だけ自由に移動できる桃鉄のロジックを実装する。

## ボードの仕様
- 6列 × 4行 = 24マス
- マスの種類:
  - 通常マス
  - 🏁 ゴールマス（1つ）

## プレイヤーの仕様
- 1人
- 初期位置: 任意のマス（例: 左上）

## ゲームの流れ
1. 「サイコロを振る」ボタンを押す
2. 1〜6がランダムに出る
3. 出た目の合計数だけ十字キーで1マスずつ移動できる
4. 残り移動数が0になったらターン終了

## 操作仕様
- ↑↓←→ キーで1マスずつ移動
- 移動するたびに残り移動数が1減る
- 端のマスでその方向キーを押しても何も起きない（残り移動数も減らない）

## 画面の仕様
- 6×4のボード表示
- プレイヤーの現在位置のハイライト
- サイコロの結果表示
- 残り移動数の表示

## 今回のスコープ外

- マスの効果（お金の増減）
- 勝敗判定
- CPU対戦
